# Lab 3D Scene

A React Three Fiber project recreating a laboratory scene in a fine vector style.

## Setup

1. Open a terminal in this directory: `d:\lab-3d-scene`
2. Install dependencies:
   ```bash
   npm install
   ```
3. Run the development server:
   ```bash
   npm run dev
   ```

## Project Structure

- `src/components/`: Contains all 3D models (Bench, Hood, Stool, Room, Lights).
- `src/App.jsx`: Main scene setup with lighting and camera.
- `src/LabScene.jsx`: Composition of the room.

## Features

- **Vector Style Aesthetics**: Clean geometry, soft lighting, and minimal textures.
- **Modular Components**: Easily reusable lab furniture.
- **React Three Fiber**: Declarative 3D scene.
